p1089-津津的储蓄计划

P1089 [NOIP 2004 提高组] 津津的储蓄计划

题目描述

津津的零花钱一直都是自己管理。每个月的月初妈妈给津津 $300$ 元钱，津津会预算这个月的花销，并且总能做到实际花销和预算的相同。

为了让津津学习如何储蓄，妈妈提出，津津可以随时把整百的钱存在她那里，到了年末她会加上 $20%$ 还给津津。因此津津制定了一个储蓄计划：每个月的月初，在得到妈妈给的零花钱后，如果她预计到这个月的月末手中还会有多于 $100$ 元或恰好 $100$ 元，她就会把整百的钱存在妈妈那里，剩余的钱留在自己手中。

例如 $11$月初津津手中还有 $83$ 元，妈妈给了津津 $300$ 元。津津预计$11$月的花销是 $180$ 元，那么她就会在妈妈那里存 $200$ 元，自己留下 $183$ 元。到了 $11$ 月月末，津津手中会剩下 $3$ 元钱。

津津发现这个储蓄计划的主要风险是，存在妈妈那里的钱在年末之前不能取出。有可能在某个月的月初，津津手中的钱加上这个月妈妈给的钱，不够这个月的原定预算。如果出现这种情况，津津将不得不在这个月省吃俭用，压缩预算。

现在请你根据 $2004$ 年 $1$ 月到 $12$ 月每个月津津的预算，判断会不会出现这种情况。如果不会，计算到 $2004$ 年年末，妈妈将津津平常存的钱加上 $20%$ 还给津津之后，津津手中会有多少钱。

输入格式

$12$ 行数据，每行包含一个小于 $350$ 的非负整数，分别表示 $1$ 月到 $12$ 月津津的预算。

输出格式

一个整数。如果储蓄计划实施过程中出现某个月钱不够用的情况，输出 $-X$，$X$ 表示出现这种情况的第一个月；否则输出到 $2004$ 年年末津津手中会有多少钱。

注意，洛谷不需要进行文件输入输出，而是标准输入输出。

题目简述

  • 妈妈每个月给津津300元(共12个月)
  • 津津每个月都会给自己设立一个花销预算即huaxiao
  • 如果津津每个月的总钱去掉n还剩100或者大于100就交给妈妈
  • 妈妈会在年终以1.2倍的形式还于津津
  • 如果津津的总钱小于花销预算则输出当前月份的负数，反之输出最终总钱

题目题解

mom_total=0 
total=0 
for month in range(1, 13):
    huaxiao=int(input())
    total=total+300 
    if huaxiao>total: 
        print(-month) 
        exit()
    total=total-huaxiao 
    if total>=100: 
        borrow=total//100*100 
        mom_total=mom_total+borrow
        total=total-borrow
total=total+mom_total*1.2
print(int(total))

题目讲解

初始化变量

上文提到过’津津的总钱‘和’妈妈的总钱‘那么我们就可以知道两个初始化变量

  • total=0津津的总钱
  • mom_total=0妈妈的总钱

Note

初始化变量往往是在文章中不会直接提及的，需要一定的阅读能力和思维去在构思代码的过程中决策是否需要初始化变量，目前发现的唯一提升方式就是’多练‘

for循环12个月

  • for month in range(1, 13):文中提到了津津12个月的花销预算那么就要运用for循环对这12个数据进行遍历
  • huaxiao=int(input())输入12个月的花销预算
  • total=total+300 更新每个月的总钱数(即妈妈每个月给的300元+每个月剩下的钱)

if语句判断（判断是否花销预算大于津津的总钱数）

  • if huaxiao>total: 如果花销预算大于津津的总钱数
  • print(-month) 则输出当月月份的负数
  • exit()判断完之后结束程序

Warning

注意区分exit()和break的区别前者是直接结束程序后者则是单纯的退出循环还会机型后续的语句

刷新total值

  • total=total-huaxiao 在上文的判断之后若没有符合则跳到此行

Warning

此行不在两个if语句块里面需要注意，只是在for循环里面用来每月更新津津的总钱的值

if语句判断（判断是否需要交给妈妈钱）

  • if total>=100: 判断津津的总钱是否大于等于100
  • borrow=total//100*100这里的borrow即为给妈妈的钱数，津津的总钱取整100再乘以100
  • mom_total=mom_total+borrow更新妈妈的总钱
  • total=total-borrow给了妈妈多少钱津津的总钱自然也跟着相应的减少

Note

设一个borrow是为了让整体更具可读性，是可有可无的

最后12个月结算总钱

  • total=total+mom_total*1.2因为妈妈会在年终以1.2倍的形式还于津津，所以更新值津津的总钱加上妈妈的总钱加倍后还回
  • print(int(total))最后输出津津的总钱

Note

加int是为了让total转为整数类型是必加的，因为前面total=total+mom_total*1.2有乘以小数输出的时候如果不强制转换为整数就会出现1580.0的情况是不符合输出要求的

总结

此题运用的东西很少以至于一个函数都不用调用，适合用来练习思维能力，难度偏低
