B2043 判断能否被 3，5，7 整除

题目描述

给定一个整数 $x$，判断它能否被 $3$，$5$，$7$ 整除，并输出以下信息：

1、能同时被 $3,5,7$ 整除（直接输出 3 5 7，每个数中间一个空格）；

2、只能被其中两个数整除（按从小到大的顺序输出两个数，例如：3 5 或者 3 7 或者 5 7，中间用空格分隔）；

3、只能被其中一个数整除（输出这个除数）；

4、不能被任何数整除，输出小写字符 n。

输入格式

输入一行，包括一个整数 $x$。

输出格式

输出一行，按照描述要求给出整数被 $3$，$5$，$7$ 整除的情况。

输入输出样例 #1

输入 #1

105

输出 #1

3 5 7

说明/提示

数据规模与约定

对于全部的测试点，保证 $1 \leq x \leq 200$。

题目答案(两种)

  • 正常写法，行数较多，结构重复简单主用if与elif语句进行编程，不过多赘述
x=int(input())
if x%3==0 and x%5==0 and x%7==0:
    print('3 5 7')
elif x%3==0 and x%5==0:
    print('3 5')
elif x%3==0 and x%7==0:
    print('3 7')
elif x%5==0 and x%7==0:
    print('5 7')
elif x%7==0 or x%3==0 or x%5==0:
    print('x')
else:
    print('n')
  • 简洁写法 重点在于这个脱离了常规if写法运用了for循环和列表添加
x = int(input())
divs = []
for d in (3, 5, 7):
    if x % d == 0:
        divs.append(str(d))

if divs:
    print(" ".join(divs))
else:
    print('n')

实例演示

初始条件

  • x = int(input()) 为输入整数值
  • divs = [] 这个是重点为后续的'append'添加列表做先决条件

for循环判定

  • for d in (3, 5, 7): 设置一个变量名'd'用来在(3,5,7)这三个数里面进行检索
  • if x % d == 0: if条件判断‘x’输入值是否能被‘d’所在的三个数整除
  • divs.append(str(d)) 如果上面的if判断成立，则在'divs'这个空列表内添加能被整除的数

Note

如果不成立则'd'在(3,5,7)内再往后检索一位直至跳出for循环，若都不成立则无输出直接跳出for循环

if判断

  • if divs: 此句是在判断'divs'这个列表是否为空列表如果是则为'False'跳至else语句直接输出'n'即上句的都不成立
  • print(" ".join(divs)) 若以上if都成立则输出能被整除的数

总结

本题主要是扩展新思路，其中主要是创空列表的思路很好，再运用for循环if语句判断来做
