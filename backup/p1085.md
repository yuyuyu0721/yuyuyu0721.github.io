# P1085 [NOIP 2004 普及组] 不高兴的津津

## 题目描述

津津上初中了。妈妈认为津津应该更加用功学习，所以津津除了上学之外，还要参加妈妈为她报名的各科复习班。另外每周妈妈还会送她去学习朗诵、舞蹈和钢琴。但是津津如果一天上课超过八个小时就会不高兴，而且上得越久就会越不高兴。假设津津不会因为其它事不高兴，并且她的不高兴不会持续到第二天。请你帮忙检查一下津津下周的日程安排，看看下周她会不会不高兴；如果会的话，哪天最不高兴。

## 输入格式

输入包括 $7$ 行数据，分别表示周一到周日的日程安排。每行包括两个小于 $10$ 的非负整数，用空格隔开，分别表示津津在学校上课的时间和妈妈安排她上课的时间。

## 输出格式

一个数字。如果不会不高兴则输出 $0$，如果会则输出最不高兴的是周几（用 $1, 2, 3, 4, 5, 6, 7$ 分别表示周一，周二，周三，周四，周五，周六，周日）。如果有两天或两天以上不高兴的程度相当，则输出时间最靠前的一天。

## 输入输出样例 #1

### 输入 #1

```
5 3
6 2
7 2
5 3
5 4
0 4
0 6
```

### 输出 #1

```
3
```

# 题解(1)
最为常规且简洁的解法较为推荐此解法，主要运用了for循环语句的检索让输入语句的值进行正常运算，详细思路将在下文讲解
```python
unhappy_day=0
hour=0
for day in range(1,8):#将输入语句放入for循环里面是因为输入的格式特殊需要用for循环对day进行检索，同时让输入语句到下一数字
    mom,school=map(int,input().split())
    total=school+mom
    if total>8 and total>hour:
        hour=total
        unhappy_day=day
print(unhappy_day)
```

# 题解(2)

```python
def unhappy(x,y):#计算不高兴值公式
    a=x+y
    return max(a - 8, 0)
day=[]#创建空列表来进行每天数据的存储
for i in range(7):#获取输入的数值并将数值填入列表中
    school,kewai=map(int,input().split())
    day.append(unhappy(school,kewai))
first_unhappy=0#创建计数器用于记录哪天最不开心
for i in range(7):#历遍day列表里面的值，选出最大的值保存
    if day[i]>day[first_unhappy]:#如果当前的不开心值大于first_unhappy
        first_unhappy=i#以最大值替换掉当前的值
if max(day) == 0:#输出做判断
    print(0)
else:
    print(first_unhappy + 1)#列表索引从0开始的，所以要加一位
```

# 题解(3)
```python
# 用列表存储7天的总时长（索引0对应周一，索引6对应周日）
total_hours = []

# 读取7天的数据并计算总时长，存入列表
for _ in range(7):
    school, mom = map(int, input().split())
    total = school + mom
    total_hours.append(total)

# 初始化最大时长和对应星期（默认0表示不不高兴）
max_total = 0
result_day = 0

# 遍历列表，寻找最不高兴的日子
for i in range(7):
    # 当前天的总时长超过8小时，且比之前记录的最大时长更长
    if total_hours[i] > 8 and total_hours[i] > max_total:
        max_total = total_hours[i]
        # 星期是索引+1（因为索引0是周一）
        result_day = i + 1

print(result_day)
```

# 题解(1)讲解

## 先决条件
* `unhappy_day=0`记录最不高兴的一天
* `hour=0`记录最不高兴的一天的具体学习时长

## for循环语句
* `for day in range(1,8):`用day在1到8(不包括8，原因看下文)里面进行检索，分别代表一到星期七
*  `mom,school=map(int,input().split())`输入`mom`和`schoo`两个变量名分别代表妈妈给的学习时长和学校给的学习时长
* `total=school+mom`总和的学习时长为学校的加上妈妈的

## if语句判断嵌套
* `if total>8 and total>hour: `判断如果学习时长总和大8并且学习时长总和大于最不高兴的一天的具体学习时长即时长相同但天数更早则执行其下语句块
* `hour=total`当当前的学习时长大于最高纪录时长的时候最不高兴的一天的具体学习时长等于总和的学习时长
* `unhappy_day=day`记录最不高兴的一天等于当前星期几(为后面的输出做先决条件)

## 输出
* `print(unhappy_day)`输出最不高兴的一天

# `range`讲解
`range`常用于for循环之中表示范围
* 格式为`range()`
* 当括号内为一个参数时`range('要循环的次数')`
* 当括号内为两个参数时`range('起始数字，结束数字')`
>[!NOTE]
当括号内为两个参数时结束数字是不包括本身的而初始数字则是包括，简称包头不包尾
>

* 当括号内为三个参数时`range('起始数字，结束数字，步长') `

# 题解(2)讲解
>[!WARNING]
此法运用def函数和for                                                                                                                                                                                                                                                                                                           循环运行，难度较为高
>

## def函数定义('整体主要计算不高兴值公式')
* `def unhappy(x,y)`两个输入值参数为后面的输入变量的先决条件
* `a=x+y`计算表学习时长的总和
* `return max(a - 8, 0)`返回`(a - 8, 0)`中的最大值来筛选大于8小时学习时长的星期

## 定义
* `day=[]`创建空列表来进行每天数据的存储为后面的添加先决条件

## for第一次循环('主要作用:获取输入的数值并将数值填入列表中')
* `for i in range(7):`让`i`循环七次
* `school,kewai=map(int,input().split())`获取输入的数值并将数值填入 `def unhappy(x,y)`
* `day.append(unhappy(school,kewai))`将数值填入列表中

## 定义
* `first_unhappy=0`创建计数器用于记录哪天最不开心

## 第二次for循环('主要作用:历遍day列表里面的值，选出最大的值保存')
* `for i in range(7):`检索
* `if day[i]>day[first_unhappy]:`如果当前的不开心值大于`first_unhappy`，即`day[i]`为列表里的当前学习时长`day[first_unhappy]`为历史最高学习时长，`day[i]`中的`i`会根据上文的for循环而改变，`day[first_unhappy]`则会根据if语句判断改变
* `first_unhappy=i`以最大值替换掉当前的值

## if语句判断
* `if max(day) == 0:`及`print(0)`如果列表为空即没有不高兴的时候则输出'0'

## else
* `print(first_unhappy + 1)`加1是因为上文的索引索引是从0开始的，即`day[0]`时输出的则是第一天的学习时长1的时候则为第二天，而题目要求我们转换为星期则加1
