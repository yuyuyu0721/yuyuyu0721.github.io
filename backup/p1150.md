# P1150 Peter 的烟
## 题目描述
Peter有n根烟，k（k>1)个烟蒂可以换一个新烟
`输入格式 一行包括两个整数n,k(1<n,k<10**8)`
`输出格式一个整数包括最终烟的根数`
```python
n, k = map(int, input().split())
total = n  # 初始的烟
butts = n  # 初始的烟蒂

# 当烟蒂数量足够兑换新烟时
while butts >= k:
    # 计算能换多少根新烟
    new_cigarettes = butts // k
    total += new_cigarettes  # 加上新换的烟

    # 更新烟蒂数量：兑换后剩余的烟蒂 + 新烟吸完产生的烟蒂
    butts = (butts % k) + new_cigarettes

print(total)
```
>[!NOTE]
本题重在理解新烟和新烟蒂的变化(要能想到将n初始的输出值改为两个变量)即以下所展示的代码
前者用自然语言来讲就是新的烟等于初始的烟蒂整除烟蒂可换取一个新烟的数量
后者则是新烟蒂数量等于烟蒂取余初始的烟蒂(兑换后剩余的烟蒂)整除烟蒂可换取一个新烟的数量加新烟吸完产生的烟蒂
`new_cigarettes = butts // k和 butts = (butts % k) + new_cigarettes` 
### 举个例子
设初始的烟和烟蒂为4，k值输入为4(则四个烟蒂换一个新烟)
符合while循环中的 'butts >= k'则 new_cigarettes=4//4=1
总和记录=1(新烟）+4（初始的烟）
最后更新烟蒂数量=兑换后剩余的烟蒂(这里即为0)+新烟吸完后产生的新烟蒂(这里即为1)
>
### 总结
本题在做的时候重在能想到将一个输出名改为两个变量名 如果是单纯理解代码则重在理解新烟和新烟蒂的变化
主要使用while循环和运算符运用
