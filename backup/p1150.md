# P1150 Peter 的烟
## 题目描述
Peter有n根烟，`k（k>1)`个烟蒂可以换一个新烟
输入格式 一行包括两个整数`n,k(1<n,k<10**8)`
输出格式一个整数包括最终烟的根数
```python
n, k = map(int, input().split())
total = n  # 初始的烟
butts = n  # 初始的烟蒂
#当烟蒂数量足够兑换新烟时
while butts >= k:
    # 计算能换多少根新烟
    new_cigarettes = butts // k
    total += new_cigarettes  # 加上新换的烟
    # 更新烟蒂数量:兑换后剩余的烟蒂 + 新烟吸完产生的烟蒂
    butts = (butts % k) + new_cigarettes
print(total)
```
> [!NOTE]
本题重在理解新烟和新烟蒂的变化(要能想到将n初始的输出值改为两个变量)即以下所展示的代码
`new_cigarettes = butts // k`和 `butts = (butts % k) + new_cigarettes` 
>

### 举个例子
## 初始条件
*`k=4`:兑换比例为四个烟蒂换一个新烟
*`total=4`:初始的烟为4
*`butts=4`:抽完初始的烟后产生的烟蒂为4
## while循环条件判断
*`butts>=k`:判断烟蒂数量是否大于或者等于烟与烟蒂的兑换比例(即是否能换新烟）
## while循环条件符合执行其中语句块
*兑换的新烟:` new_cigarettes = butts // k =4//4=1`
*兑换完后烟的总数:`total += new_cigarettes=1(新烟）+4（初始的烟）=5`
*更新烟蒂数量:` butts = (butts % k) + new_cigarettes`兑换后剩余的烟蒂+新烟吸完后产生的新烟蒂=0+1=1
### 总结
本题在做的时候重在能想到将一个输出名改为两个变量名 如果是单纯理解代码则重在理解新烟和新烟蒂的变化
主要使用while循环和运算符运用
